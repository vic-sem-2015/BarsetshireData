Data and tools for preliminary analysis of Anthony Trollope's Barsetshire Series by the 2015 Victorian Novel Research Seminar. Includes cleaned Project Gutenberg plain texts of the six novels in the series, directories of individual chapters for each novel (for use with MALLET topic modeling software),  Python scripts used to split chapters, stopword lists customized for the Barsetshire series, lists of named entities extracted using the Stanford Named Entity recognizer, and notes from brainstorming discussions. You can play with [topic models of the Barsetshire novels here](http://rachelsagnerbuurma.org/Barsetshire/topics.html). We used [David Mimno's](https://mimno.infosci.cornell.edu) [implementation of Latent Dirichlet Allocation in Javascript and D3](https://mimno.infosci.cornell.edu/jsLDA/index.html).
